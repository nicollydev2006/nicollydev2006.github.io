## ✨ Funcionalidades

- **Boot screen animada** — simulação de terminal ao carregar a página
- **Command Palette** (`Ctrl+K`) — navegação rápida por teclado com filtro em tempo real
- **Recruiter Mode** — modal com informações essenciais para recrutadores
- **Idle screen** — tela de standby após inatividade
- **Easter egg** — Konami Code ativado com ↑↑↓↓←→←→BA 🎮
- **Toast de confirmação** — feedback visual ao copiar e-mail ou número
- **Design responsivo** — funciona em desktop e mobile
