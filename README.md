## 🚀 Features

| Feature | Descrição |
|---|---|
| **Boot Screen** | Animação de terminal ao carregar a página |
| **Cursor Personalizado** | Dot + ring com efeito hover em amarelo |
| **Command Palette** | Atalho `Ctrl+K` para navegação rápida |
| **Recruiter Mode** | Modal com dados completos para recrutadores |
| **Scroll Reveal** | Animações suaves ao rolar a página |
| **Partículas + Matrix** | Canvas animado com código caindo no fundo |
| **Idle Screen** | Tela de pausa após 45s inativo |
| **Konami Code** | Easter egg secreto (↑ ↑ ↓ ↓ ← → ← → B A) |
| **Toast Notifications** | Feedback visual ao copiar e-mail/telefone |
| **Totalmente Responsivo** | Adaptado para mobile |
